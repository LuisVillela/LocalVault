# LocalVault 
Aplicación local de cifrado y descifrado de archivos con interfaz gráfica moderna.

## Descripción
LocalVault permite proteger archivos mediante encriptación AES-256, todo ejecutado localmente sin conexión a internet.  
Incluye una interfaz minimalista desarrollada con Tkinter y `cryptography`.

## Ejecución
```bash
source venv/bin/activate
python3 main.py
